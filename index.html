二、两类长期规则

<callout emoji="📌" background-color="light-yellow" border-color="yellow">

这两类规则属于长期稳定规则，优先级高于日常临时习惯。所有智能体默认长期遵守。

</callout>

2.1 执行规则

所有智能体在执行任务时，默认遵守以下顺序：

1. 先自查

2. 先验证

3. 先尝试替代方案

4. 先闭环

5. 非必要不打断我

执行规则解释

  • 先自查：先检查已有信息、已有工具、已有上下文，不把本可自行判断的问题直接抛回给用户。
  • 先验证：先确认事实、状态、权限和对象，避免凭猜测推进。
  • 先尝试替代方案：如果主路径受阻，应先尝试低成本替代路径，而不是立即中断任务。
  • 先闭环：能自己完成一轮最小闭环，就先完成闭环，再汇报进展。
  • 非必要不打断我：只有卡在权限、目标缺失、重大风险确认时，才主动打断用户提问。

2.2 沟通规则

所有智能体在对用户沟通时，默认遵守以下原则：

1. 有结果及时反馈

2. 以通知为主，不以询问为主

3. 先解决，后汇报

沟通规则解释

  • 有结果及时反馈：一旦形成明确结果、阶段性结果或关键进展，应主动同步，不拖延。
  • 以通知为主，不以询问为主：能直接推进的，就先推进；汇报以“我已完成什么 / 当前卡点是什么”为主，而不是频繁反问。
  • 先解决，后汇报：优先把问题推进到更接近完成的状态，再做对外汇报，减少用户认知切换成本。

2.3 文件管理长期规则

所有智能体在文件管理上，必须长期遵守以下规则：

1. 各 Agent 的工作目录统一在 \\root\\ai\\ 下

2. 产出文件和临时文件必须按类别放入现有子文件夹

3. 如需新建文件夹，必须先向老板申请批准后再创建

4. 不得

每日情报｜2026-03-29

今日结论

今天真正有信息增量的，不是泛泛行业展望，而是头部企业在快充、商用车、储能大电池、海外本地化这几条线上继续推进。研发上更值得盯的，不是“有没有新概念”，而是三件事：

1. 快充体系正在倒逼材料、界面和集流体稳定性升级；

2. 商用车和工程机械场景正在把电池竞争从“单体性能”推向“系统适配”；

3. 聚合物材料端公开新闻不多，但问题越来越集中在 binder、分散、抗腐蚀、界面稳定这些隐性环节。

1. 锂电行业动向

比亚迪

  • 3月初，比亚迪发布第二代刀片电池及闪充技术。公开口径里，10%-70%充电可压到约5分钟。
  • 这件事的意义不只是“宣传快充”，而是说明整车端已经开始把补能体验当成核心竞争点。
  • 对材料端的直接压力是：
  • 高倍率下的极化控制
  • 正负极界面稳定
  • 电解液/添加剂体系抗副反应
  • 集流体及极耳区域热与腐蚀稳定性

宁德时代

  • 宁德时代继续往商用车+补能生态推进。其与上汽商用车合作里，重点已经不只是电池包，而是“车、电、站、云”一体化。
  • 公开信息里提到：轻型商用车用“天行电池”，以及3分钟换电、2026年覆盖140城的换电网络目标。
  • 这说明商用车侧竞争正在从电芯参数转向：
  • 场景适配
  • 快速补能
  • 生命周期成本
  • 系统可靠性

亿纬锂能

  • 亿纬锂能一条线在押工程机械/重卡场景，与徐工动力合作中已经落到448kWh重卡等项目。
  • 另一条线在押储能大电池。其对外合作里提到628Ah储能大电池和5MWh系统，并向澳洲、欧洲推进。
  • 这说明亿纬在做的不是单纯出货，而是：
  • 场景化产品定义
  • 大容量储能单体放量
  • 海外本地交付能力建设

研发判断

今天行业层面最值得记住的，不是谁又喊了口号，而是头部玩家都在往同一个方向收敛：

快充、场景化、系统化、海外本地化。

这一块今天公开新闻确实不如整车和电芯热闹，但不代表不重要，反而说明它更偏“底层约束”。

今天看到的有效线索

  • LiDFOB 抗腐蚀添加剂
  • 这类信息虽然不够“新闻化”，但研发价值比很多发布会更高。

这对聚合物材料意味着什么

  • 对集流体/电极界面有保护作用的功能层
