八、优先级推荐（针对本项目目标）

目标：100℃基本不动 / 120–200℃快速起泡 / 高产气率 / 体系可调

第一梯队（优先做）

编号方案理由
A1ADCA + 活化剂（ZnO/硬脂酸锌）产气量高；温区可调；工艺最成熟；最可能兼顾稳定性与起泡速度
A2OBSH 单剂中温区间代表；快速起泡；作 ADCA 对照

第二梯队（补充验证）

编号方案理由
B1ADCA + 活化剂（尿素/水杨酸）拉低活化温度；对比不同活化机制
B2TSH 单剂低温快速触发对照
B3包覆 NaHCO₃ + 潜伏酸源低成本路线验证；适合做对比基线

不建议优先做

方案原因
AIBN、ABVN100℃稳定性无解
裸混 NaHCO₃ + 有机酸储存稳定性差；无法工程化
3组分以上复配第一轮筛选阶段不应引入复杂变量

九、关键验证指标清单

热稳定指标

  • 100℃保温 1h / 2h 失重（必须 < 5%，否则淘汰）
  • 100℃保温后颜色、气味、流变变化
  • 储存 7 天后再测 DSC/TGA（对比初始样）

发泡指标

  • 起泡起始温度（TGA/DTG）
  • 主发泡峰温
  • 发泡完成温度
  • 发泡速率（等温产气测试）
  • 最大膨胀倍率（基体内验证）

产气指标

  • 理论产气量 vs 实际产气量（密闭釜法）
  • 目标温区（120–200℃）释放完成度

结构指标

  • 泡孔尺寸、分布、均匀性
  • 开孔/闭孔比例
  • 塌泡倾向

残留指标

  • 灰分/残渣量
  • 离子残留
  • 气味、黄变/色变

第一轮：TGA 筛选（8–10 个样品）

样品编号配方测试项目

F-02ADCA + ZnO（1 phr）TGA + DTG

F-04ADCA + 硬脂酸锌（3 phr）TGA + DTG
F-05ADCA + 尿素（5 phr）TGA + DTG
F-06OBSH 纯品TGA + DTG
F-07OBSH + 稳定剂（0.5 phr）TGA + DTG
F-08TSH 纯品TGA + DTG
F-09包覆 NaHCO₃ + PTSTGA + DTG
F-10包覆 NaHCO₃ + 邻苯二甲酸酐TGA + DTG

所有样品必做：100℃等温 TGA 1h（测稳定性）

第二轮：实际产气验证（通过第一轮筛选的样品）

  • 密闭釜法：120℃ / 160℃ / 190℃ 三档
  • 收集总产气量，与理论值对比
  • 优先测 160℃ 和 190℃ 两档

第三轮：基体内验证（通过第二轮的样品）

  • 配入基体（树脂/橡胶/塑料）
  • 130℃ / 160℃ / 190℃ 三档模压或挤出
  • 评价：起泡倍率、泡孔均匀性、残留影响

⚠️ 研发判断：没有验证路径的判断不算完整判断。上述表格中的"推荐"仅为初筛方向，最终结论必须以实验数据为准。
