# 2026-03-30 linux.do 授权公益站查询与签到任务

## 任务主题
linux.do 授权公益站，查询

## 任务目标
1. 拿到授权关联的所有公益站的 API 调用信息和密钥。
   - 数据来源：当前授权列表页面（可能有重复）
   - 页面：<https://connect.linux.do/oauth2/log>
2. 通过 API 和密钥，查询各公益站的模型列表与可用状态。
3. 如果公益站有签到功能，尝试编写签到脚本。

## 当前理解
- 这是一个调查 + 自动化实现任务。
- 优先产出：
  1. 授权站点清单
  2. 每个站点的 API / Key / 模型状态调查结果
  3. 可行时给出签到脚本 MVP

## 建议执行顺序
1. 登录并读取授权日志页面
2. 提取并去重公益站列表
3. 逐站分析 API 接入方式与密钥获取方式
4. 验证模型列表查询接口和可用性
5. 识别是否存在签到入口/接口
6. 为可签到站点编写脚本

## 风险与限制
- 可能需要登录态
- 部分站点可能没有公开 API 文档
- 部分密钥可能无法从授权页直接获取
- 签到逻辑可能依赖前端签名、CSRF 或人机校验

## 交付预期
- 一份整理后的调查结论
- 可运行的查询脚本 / 签到脚本（MVP 优先）
