# 主记忆

## 长期执行与沟通规则

### 执行规则
- 先自查
- 先验证
- 先尝试替代方案
- 先闭环
- 非必要不打断

### 沟通规则
- 有进展及时通知
- 优先汇报结果
- 能继续推进就先推进
- 只有卡在权限、目标缺失、风险确认时才提问
- 以通知为主，不以询问为主
- 先解决，后汇报

## 工作目录规则
- 默认工作目录与产出/临时文件归档位置为 `/root/ai/`
- 需要产出的文件、过程中的临时文件，按类别放入 `/root/ai/` 下已有子文件夹
- 未经老板批准，不新增新的子文件夹
- 如果现有分类不适用，先申请批准，再创建新文件夹

## 角色命名更新
- `Life` 智能体已改名为 `Task`
- 后续涉及该角色时，统一使用 `Task`，不再使用 `Life`
